# idea功能面板介绍

- File –  主要是对文件的操作和设置
- Edit – 对文本的一些操作
- Navigate – 项目工程之间的操作
- Code – 对代码的一些操作

- Analyze – 对源码分析和项目依赖的一些操作
- Refactor – 代码重构的一些操作
- Build – 构建工程
- Run – 运行
- Tools – 工具
- VCS – 版本控制

# 文件操作

- 查看最近打开的文件
  - Ctrl + E
- 查看最近修改的文件
  - Alt +Shift + C
- 查找类
  - Ctrl + N
- 查找文件
  - Ctrl +Shift + N
  - 按两次 Shift
- 全局搜索
  - Ctrl +Shift + F

# 书签

- 添加/取消书签
  - Ctrl + F11
- 查看书签
  - Shift + F11
- 跳转到书签位置(仅限于数字书签)
  - Ctrl + 书签数字标志
